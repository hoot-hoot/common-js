# Common [![Build Status](https://travis-ci.org/hoot-hoot/common-js.svg?branch=master)](https://travis-ci.org/hoot-hoot/common-js) [![Coverage](https://codecov.io/gh/hoot-hoot/common-js/branch/master/graph/badge.svg)](https://codecov.io/gh/hoot-hoot/common-js)

A bunch of basic type definitions and utility functions, meant to be used across `hoot-hoot` JavaScript/TypeScript code. This package contains only isomorphic code, so it can be used in both a client and server context.
